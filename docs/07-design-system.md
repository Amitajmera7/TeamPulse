# 🎨 TeamPulse Design System

> Version 1.0

---

# Purpose

This document defines the visual language of TeamPulse.

Every page, component and interaction must follow this design system to ensure consistency and a premium SaaS experience.

---

# Design Philosophy

TeamPulse should feel like a modern B2B SaaS analytics platform.

Inspired by:

- Linear
- Stripe
- Vercel
- Atlassian Analytics
- GitHub

Design Principles:

- Clean
- Minimal
- Spacious
- Professional
- Fast
- Data-first

---

# Spacing

Use an 8px spacing system.

| Token | Value |
|--------|-------|
| xs | 4px |
| sm | 8px |
| md | 16px |
| lg | 24px |
| xl | 32px |
| xxl | 48px |

Never use arbitrary spacing values.

---

# Border Radius

Cards

16px

Buttons

12px

Inputs

10px

Tags

999px

---

# Shadows

Use subtle elevation.

Avoid heavy shadows.

Cards should appear light and modern.

---

# Icons

Use Lucide React.

Guidelines:

- Consistent stroke width
- Avoid mixing icon libraries
- Pair icons with labels where appropriate

---

# Buttons

Primary

Filled

Blue

Secondary

Outlined

Ghost

Text only

Danger

Red

Loading state required.

---

# Cards

All cards should include:

- Title
- Value
- Optional subtitle
- Optional trend
- Hover effect
- Responsive layout

---

# Tables

Requirements:

- Sticky header
- Hover rows
- Sorting
- Search
- Pagination
- Empty state

---

# Charts

Preferred chart types:

- Line
- Area
- Bar
- Stacked Bar
- Heatmap
- Donut

Avoid unnecessary pie charts.

---

# Animations

Use subtle animations only.

Examples:

- Card hover
- Counter animation
- Chart transitions
- Smooth page transitions

Animation duration:

150–300ms

---

# Responsive Breakpoints

| Device | Width |
|----------|------|
| Mobile | <640px |
| Tablet | 640–1024px |
| Desktop | >1024px |

---

# Empty States

Every page must explain:

- Why no data exists
- What the user can do next

---

# Loading States

Use Skeleton Loaders.

Never display blank screens.

---

# Accessibility

- Keyboard navigation
- Focus indicators
- Screen reader labels
- Color contrast
- Tooltips
