# 🚀 TeamPulse

> **AI-Powered Engineering Intelligence Platform**

Transform Jira activity into actionable engineering insights.

---

## 📖 Overview

TeamPulse is a modern Engineering Intelligence Platform that helps Engineering Managers, Delivery Managers and CTOs understand engineering performance using Jira data.

Instead of displaying only ticket counts and worklogs, TeamPulse provides meaningful engineering KPIs, delivery trends, workload analytics and AI-powered insights.
